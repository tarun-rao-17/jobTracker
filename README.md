# Job Application Tracker (MEAN Stack)

A full-stack **Job Application Tracking System** built using the **MEAN stack** with secure authentication and a modern Angular frontend.

---

## 🚀 Features

### 🔐 Authentication
- User Registration & Login
- JWT authentication using **HTTP-only cookies**
- Protected routes (backend + frontend guard)

### 📄 Job Management
- Create job applications
- View all applications (user-specific)
- Update job status (Applied / Interview / Offer / Rejected)
- Delete job applications
- Search jobs by company name
- Filter jobs by status
- Dashboard statistics

---

## 🛠 Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- bcryptjs
- Cookie-based auth

### Frontend
- Angular (Standalone Components)
- HttpClient
- Route Guards
- FormsModule
- Cookie-based authentication
